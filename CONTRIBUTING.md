# Contributing

Thanks for helping improve the MakerRelay community space.

## Issues And Discussions

Use issues or discussions for:

- Feature requests
- Bug reports
- Listing corrections
- Spam or unsafe listing reports
- Public roadmap discussion

Do not include private information, secrets, credentials, or sensitive security details in public issues.

## Pull Requests

Pull requests are welcome only for improvements to this repository's public documentation, issue templates, or community guidelines.

Examples of acceptable pull requests:

- Fixing typos in the README
- Improving issue template wording
- Clarifying the code of conduct
- Improving public community instructions

Do not submit MakerRelay application code changes in this repository.

## Private Reports

For private or sensitive reports, email support@makerrelay.com.
