# MakerRelay Community

This repository is the public community hub for MakerRelay.

MakerRelay is a discovery directory for AI tools, Claude skills, MCP servers, CLI apps, side projects, and useful GitHub repositories.

Use this repository to:

- Request product features
- Report website bugs
- Suggest listing corrections
- Report spam, unsafe links, or misleading listings
- Follow public roadmap discussion

For private or sensitive reports, do not open a public issue. Email support@makerrelay.com instead.

## Important Links

- Website: https://makerrelay.com/
- Submit a project: https://makerrelay.com/submit
- GitHub Trending: https://makerrelay.com/trending
- FAQ: https://makerrelay.com/faq

## Open An Issue

Choose the issue type that matches what you need:

- Feature request: suggest a user-facing improvement.
- Bug report: report something broken on MakerRelay.
- Listing correction: fix inaccurate listing metadata or broken project links.
- Spam or unsafe listing: report abuse, malware, phishing, or misleading submissions.

Please include public evidence when reporting listing problems. MakerRelay may update, hide, remove, or leave a listing unchanged depending on the evidence and service policies.
